Petal Graph Search Visualizer

An interactive graph search system built with Python that lets you visualize how different search algorithms explore a graph in real time.

 Features
- 6 search algorithms: BFS, DFS, IDS, UCS, Greedy, A*
- Add nodes, edges, and weights interactively
- Heuristic input for informed search (Greedy & A*)
- Real-time graph visualization with path highlighting
- Shows visited nodes, final path, and total cost

Libraries Used
 tkinter - GUI
 matplotlib - graph visualization
 networkx - graph data structure

How to Run

1. Install dependencies:
pip install matplotlib networkx


2. Run the project:
pip install -r requirements.txt
